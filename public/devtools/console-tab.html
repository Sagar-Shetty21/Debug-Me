üìã Before You Start

Open your browser's Developer Tools (F12 or Ctrl/Cmd+Shift+I) and navigate to the Console tab. All debugging output and examples will appear there!

1. Basic Console Methods

Learn the fundamental console methods for different types of output:

Basic Logging

console.log('Hello World!') console.info('Information message') console.warn('Warning message') console.error('Error message')

Variable Inspection

const user = { name: 'John', age: 30 } console.log('User:', user) console.dir(user) // Detailed object view

2. Advanced Console Features

Grouped Logging

console.group('User Actions') console.log('Login attempt') console.log('Data fetch') console.groupEnd()

Table Display

const users = [ { name: 'John', age: 30 }, { name: 'Jane', age: 25 } ] console.table(users)

Performance Timing

console.time('operation') // Some operation console.timeEnd('operation')

Assertions

console.assert(5 > 3, '5 is greater than 3') console.assert(2 > 5, '2 is NOT greater than 5')

4. Error Debugging Scenarios

Reference Error

// Accessing undefined variable console.log(undefinedVariable)

Type Error

// Calling method on null/undefined const obj = null obj.someMethod()

Syntax Error

// Invalid JSON parsing JSON.parse('{ invalid: json }')

Custom Error Handling

try { riskyOperation() } catch (error) { console.error('Caught:', error) }
